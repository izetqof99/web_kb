# Sistem Informasi Stunting

## Deskripsi
Aplikasi berbasis web untuk pendataan ibu hamil dan anak stunting.

## Teknologi
- PHP Native
- MySQL
- HTML, CSS

## Cara Menjalankan
1. Import database dari folder /database
2. Jalankan di localhost (XAMPP/Laragon)
3. Akses melalui browser
